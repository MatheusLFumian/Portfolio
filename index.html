Sobre Mim

Nasci e cresci na pequena e pacata cidade de Tombos (MG). Com 18 anos me mudei pra Volta Redonda (RJ) para cursar Engenharia Mecânica na UFF. E foi lá que tive meu primeiro contato com programção.

Comecei aprendendo a lógica e, depois de quebrar um pouco a cabeça, comecei a gostar da ideia de desenvolver soluções para problemas que iria enfrentar diversas vezes. Um pouco depois comecei a estudar o Fortran 95, linguagem "morta" mas que tem grande aplicação matemática.

Alguns anos após ter começado a faculdade de engenharia, me toquei que aquela não era minha praia. Migrei então para o curso de Tecnologia em Sistemas da Informação, também pela UFF. Conheci então o Python, linguagem que me conquistou por toda sua capacidade no desenvolvimento web, desktop e a aplicação em ciência de dados. Iniciei meus estudos também em HTML e CSS

Em Setembro de 2019 comecei a estagiar e logo aprendi PHP. Hoje, trabalho como Desenvolvedor Fullstack na Elite Comunicação.

Bom, esse é um pequeno resumo da estrada até aqui. Agora, com licença, apresentarei um pouco do que sei nessa página.

Stacks que domino

Algumas tecnologias que já trabalhei, tenho estudado e um pouco da minha história com elas.

HTML, CSS, JavaScript

Podemos descrever esse trio como a base para o desenvolvimento de sites e aplicações mas não se engane, são ferramentas muitos poderosas e excênciais para um programador.

Meu primeiro contato com essas ferramentas foi no início da faculdade. Aprendi de forma rápida como aplicar as tags para desenvolver uma página básica e funcional.

Bootstrap

Um framework bem conhecido e poderoso que junta toda a base do HTML, CSS e JavaScript em um só lugar trazendo mais produtividade no trabalho.

Conheci o Bootstrap na entrevista para o meu primeiro estágio e, como já dominava a base do desenvolvimento web, aprendi rapidamente e com o passar do tempo me tornei cada vez mais produtivo utilizando a ferramenta.

PHP

A linguagem de programação voltada para web mais utilizada. Amada por muitos, odiada por tantos outros. O PHP é uma linguagem com uma curva de aprendizado relativamente baixa. Com pouco tempo já é possível criar aplicações pequenas para receber os dados de um formulário mas algo que muitos não percebem é sua robustez quando associada à uma arquitetura de software poderosa. Com uma arquitetura MVC a linguagem ganha um poder grande podendo ser utilizada no desenvolvimento de sistemas mais complexos.

Tive meu primeiro contato com o PHP no estágio, lia a documentação, aprendia com os sistemas utilizados na empresa e comecei a desenvolver sites e alguns sistemas um pouco mais complexos ainda no estágio. Atualmente desenvolvo sites mais simples e sistemas complexos com ela.

MySQL

Junto do PHP formam a dupla mais utilizada no Backend de aplicações web. O sistema de gerenciamento de banco de dados MySQL é uma ferramenta integrada na maioria dos servidores atuais, tem uma compreensão simples de como armazenar e tratar os dados e conta com sistemas que facilitam o nosso trabalho como o phpMyAdmin e o MySQL Workbench.

Comecei a utilizar o MySQL no estágio para armazenar os dados de leads dos clientes da agência. Com o tempo percebi o poder de buscar, analisar os leads recebidos e comparar com o andamento das campanhas de marketing. Uma ferramenta muito simples mas que pode gerar insites valiosos para as empresas.

WordPress

Finalmente chegamos no rei da internet, a ferramenta criada para facilitar a vida de muitas pessoas dando a elas o poder de criar sites completos sem ao menos escrever uma linha de código. O WordPress é uma sistema open source muito conhecido que domina a World Wide Web. Escrito em o PHP ele traz diversas funcionalidades próprias que já permitem qualquer pessoa criar um blog em instantes. Além disso, é possível desenvolver temas e plugins para uso próprio ou uso comercial e é nesse ponto que eu me interessei por essa ferramenta.

Utilizo o WordPress como um sistema de gerencimento de conteúdo para boa parte dos sites. Assim, não preciso me preocupar em reescrever um sistema assim para cada cliente, basta fazer algumas modificações, configurações e tenho um painel super intuitivo para cada um. Além disso a manutenção, publicação, e outros fatores são facilitados utilizando bem o rei da web.

Python

Uma linguagem de programação que conheci no início da faculdade de Tecnologia em Sistemas de Computação e logo me apaixonei. Sua curva de aprendizado é baixa, possui uma sintaxe simples e bibliotecas que podem fazer coisas incríveis.

Automação, desenvolvimento web e machine learning são as coisas que mais fazem os meus olhos brilharem quando penso em Python. Com poucas linhas é possível gerar um valor absurdo para sua produtividade e para o crescimento de sua empresa.

Google Data Studio

Ferramenta simples, intuitiva e extremamente poderosa, capaz de criar visualizações de dados dos clientes de forma dinâmica. Pode ser alimentada através de bancos de dados SQL, tabelas do Google Sheets e outras fontes como o próprio Google Analytics.

Como sua integração é simples com as ferramentas de rastreio de sites do Google, banco de dados e outras ferramentas que utilizamos nos sites além de ser gratuita, utilizo o Data Studio para dar aos clientes uma perspectiva de como seu negócio está fluindo.

Pretenções para o futuro

Vejo a programação como uma ferramenta e pretendo usá-la para solucionar os mais diversos problemas. Na internet eu vejo o futuro, seu negócio pode estar aqui, seus amigos, suas ferramentas de trabalho, enfim, tudo pode ser encontrado dentro do navegador. Meu objetivo é solucionar os problemas das empresas auxiliando elas a obterem mais lucro, maior crescimento e uma integração maior com a tecnologia.

Pretendo me tornar um bom desenvolvedor Web, refinar minhas habilidades e poder entregar ainda mais resultado a curto prazo. Já a longo prazo, pretendo me especializar em Data Science, ou seja, ajudar a colher e analisar os dados de hoje e dar insites e previsões com o machine learning para auxiliar nas decisões do futuro.

Acredito que o crescimento e o aprendizado devam ser um objetivo constante para todos. Jamais devemos nos esquecer de amar o próximo para construir uma sociedade melhor. Se formos 1% melhor a cada dia, amarmos 1% mais a cada instante, em breve poderemos chegar à Lua, quissá ao Céu.
